Declaración de Accesibilidad

Última actualización: mayo de 2026

♿ Nivel de conformidad AA (WCAG 2.1)

Compromiso con la accesibilidad

El Colegio Nuestra Señora de los Dolores está comprometido con hacer accesible el sitio web de la Escuela Infantil NSD de conformidad con el Real Decreto 1112/2018, de 7 de septiembre, sobre accesibilidad de los sitios web y aplicaciones para dispositivos móviles del sector público.

Estado de cumplimiento

Este sitio web es parcialmente conforme con el Real Decreto 1112/2018, con desviaciones menores debidas al uso de contenidos de terceros no completamente accesibles (fuentes tipográficas externas, mapa de ubicación embebido).

Características de accesibilidad implementadas

  • Estructura semántica con HTML5 (landmarks: header, nav, main, footer).
  • Etiquetas ARIA en elementos interactivos (menú de navegación, botones, formularios).
  • Texto alternativo en imágenes con función informativa (alt).
  • Jerarquía de encabezados correcta (h1 → h2 → h3).
  • Relación explícita entre etiquetas y campos de formulario (for/id).
  • Foco de teclado visible en elementos interactivos.
  • Soporte para la preferencia de movimiento reducido (prefers-reduced-motion).
  • Contraste de color suficiente en texto principal (ratio ≥ 4.5:1).
  • Sitio completamente usable sin JavaScript habilitado para el contenido principal.
