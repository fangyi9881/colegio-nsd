Declaración de Accesibilidad

Última actualización: mayo de 2025

Compromiso con la accesibilidad

El Colegio Nuestra Señora de los Dolores trabaja para garantizar la accesibilidad de este sitio web, conforme al Real Decreto 1112/2018 y las pautas WCAG 2.1 nivel AA.

Estado de conformidad

Este sitio web es parcialmente conforme con las WCAG 2.1 nivel AA. Las no conformidades detectadas se están corrigiendo progresivamente.
